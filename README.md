# 💍 Projeto Terra Média

Bem-vindo à **Terra Média**! Este é um projeto de desenvolvimento web front-end temático, imerso no universo de J.R.R. Tolkien, "O Senhor dos Anéis".

O site simula um portal que conecta fãs, apresenta "artefatos lendários" e permite o alistamento na Sociedade do Anel.

## 🚀 Funcionalidades

*   **Navegação Temática:** Design inspirado nas cores e fontes da Terra Média (Dourado e Preto).
*   **Carrossel Interativo:** Slide de imagens na página inicial controlado via JavaScript.
*   **Validação de Formulário:**
    *   Validação real de algoritmo de **CPF**.
    *   Máscaras automáticas para os campos de CPF e Telefone.
*   **Acessibilidade:** Integração com o widget **VLibras** em todas as páginas.
*   **Páginas Informativas:** Seções de Curiosidades, FAQ (utilizando tags `<details>`) e Trabalhe Conosco.
*   **Localização:** Página de contatos com endereços fictícios do universo e integração de mapa (iframe).
*   **Responsividade:** Layout adaptável utilizando CSS Grid e Flexbox.

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estrutura semântica (header, main, footer, nav, article).
*   **CSS3:**
    *   Variáveis CSS (`:root`) para gerenciamento de cores.
    *   Google Fonts ('Cinzel' para títulos e 'Lato' para corpo).
*   **JavaScript (Vanilla):**
    *   Manipulação do DOM.
    *   Lógica de validação (Regex e cálculo de dígitos verificadores).
    *   Event Listeners.

## 📂 Estrutura de Arquivos

*   `index.html`: Home com vitrine e carrossel.
*   `cadastro.html`: Formulário de registro com validação.
*   `contatos.html`: Endereços e mapa.
*   `curiosidades.html`, `faq.html`, `trabalhe-conosco.html`: Páginas de conteúdo.
*   `css/style.css`: Estilização global.
*   `script/script.js`: Lógica do carrossel e validações.

## orc_face Como Executar

Basta baixar os arquivos e abrir o `index.html` em qualquer navegador moderno. Não é necessária instalação de dependências ou servidor backend.
